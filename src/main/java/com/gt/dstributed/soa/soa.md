    SOA:
        一個系統擴展后需要將公共部分抽取，另外需要按照業務劃分服務，這個時候需要一個統一
    的方式進行交互，這個時候需要SOA；
        SOA指的是面嚮服務的架構，系統之間通過標準的方式進行交互；
        
        SOA的挑戰：
            1.多級調用帶來的延時問題：A->B->C 如果A調用B超時則直接抛出異常，不再調用服務C
            2.跟蹤調試困難
            3.更高的安全監測要求
            4.保證QOS(服務質量)
            5.高可用伸縮
